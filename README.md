# News_app
This repository consists of an express app that fetches data from a news API and renders it in container cards.

Express and Bootstrap are used as framework.
News API : https://newsapi.org/   for all articles published by the Wall Street Journal in the last 6 months, sorted by recent first

GET request : https://newsapi.org/v2/everything?domains=wsj.com&apiKey="Enter API Key here"

API keys can be obtained by signing up for a developer account for open source project, non-commercial project or commercial project in development..
